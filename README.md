
# React E-commerce Website

A complete E-commerce website built with React.js, featuring product management, shopping cart, authentication, and an admin dashboard.

## Features

- **Storefront**: Browse products, search, and filter by category.
- **Product Details**: View detailed product information.
- **Shopping Cart**: Add items, update quantities, remove items.
- **Authentication**: User login and registration (Simulation).
- **Admin Dashboard**: manage products (Create, Read, Update, Delete).
- **Animations**: Smooth page transitions using Framer Motion.

## Tech Stack

- React.js
- React Router DOM
- Framer Motion (Animations)
- Lucide React (Icons)
- Context API (State Management)
- LocalStorage (Data Persistence)

## Setup Instructions

1.  Install dependencies:
    ```bash
    npm install
    ```

2.  Run the development server:
    ```bash
    npm run dev
    ```

3.  Open your browser at `http://localhost:5173`.

## Folder Structure

- `src/components`: Reusable UI components (Navbar, Footer, ProductCard).
- `src/pages`: Main application pages (Home, Products, Cart, Auth, AdminDashboard).
- `src/admin`: Admin-specific components (AddProduct, EditProduct, ManageProducts).
- `src/context`: React Context for state management (Auth, Cart).
- `src/services`: API simulation service.
- `src/styles`: CSS stylesheets.
- `src/animations`: Animation configurations.
